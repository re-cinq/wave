Positioning vocabulary

Wave is a scaffolder for software factories — an agentic Lego baukasten with strong standards and a low-friction entry into the action. The webui is the bridge: always something to discover, start, or compare. Future: parallel runs across multiple models for cross-model agreement ("quasi-reliability") — e.g. a security audit that flags a finding on three providers carries more weight than one on one.

Scope of this batch

  • Dark theme only (one-shot pass).
  • Fake-real data: code-crispies repo, plausible issue titles, plausible signals.
  • SVG icons inline. No emojis (project policy).
  • No JS yet. Forms are visual only.
  • Layout/CSS prototype only — final will reuse internal/webui/templates/layout.html via the build-tagged /preview/* route.

Not yet mocked

  • Multi-model run detail (parallel comparison view in full).
  • Schedule editor.
  • Settings / forge auth.
  • Admin section (pipelines / personas / contracts power-user pages).
